- 🎲 **Play-to-Earn (P2E) Games**

## 🛠️ Technologies Used

- Game Engines: Unity, Unreal Engine  
- Blockchain: Solidity, Rust, BNB Smart Chain  
- Web3 Stack: Web3.js, Ethers.js, IPFS  
- Backend: Firebase, Supabase  
- AI & Automation: GPT, NPC logic scripting (planned)
